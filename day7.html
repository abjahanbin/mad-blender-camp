Working with Mixamo for Character Animation

Animation

Mixamo is a free tool from Adobe that auto-rigs 3D characters and applies motion capture animation in minutes. Upload a character mesh, choose from hundreds of pre-built animations — walk cycles, dances, combat moves — and bring it back into Blender ready to render. Full character animation without building a rig from scratch.

Mixamo mixamo.com Free character auto-rigging and motion capture library from Adobe. Sign in with a free Adobe account.
Step 1 — Export from Blender
01
Pose your character in a T-pose
Mixamo's auto-rigger works best from a T-pose — arms out to the sides, body straight, feet together. If your character is already posed differently, straighten it out before exporting.
02
Select the mesh only
Click to select only your character mesh. Leave out cameras, lights, and any existing armature. Mixamo will create its own rig — you don't need to send one.
Left Click to select mesh
03
Export as FBX
Go to File → Export → FBX. In the export options, check Selected Objects Only. Set Forward to -Z Forward and Up to Y Up. These match Blender's coordinate system to what Mixamo expects.
File → Export → FBX (.fbx)
04
Apply transforms before exporting
In the FBX export panel, enable Apply Transform. This bakes your object's position and scale into the mesh so it arrives in Mixamo at the right size and orientation.
FBX Export → Apply Transform ✓
Step 2 — Auto-Rig on Mixamo
01
Upload your character
On mixamo.com, click Upload Character and select your FBX file. Mixamo will process it and show you a preview — if the mesh looks wrong, check your export settings.
02
Place the rig markers
Mixamo shows five dots to place: chin, both wrists, and both elbows. Drag each dot to the correct joint on your character. The more accurate the placement, the better the auto-rig.
03
Let Mixamo rig it
Click Next — Mixamo generates a full skeleton and binds it to your mesh. Check Use Symmetry if your character is symmetrical. Review the result and re-place markers if any joint looks off.
04
Pick an animation
Browse the animation library on the left. Click any motion capture clip to instantly preview it on your character. Use the In Point and Out Point sliders to trim to just the section you need.
05
Download
Click Download. Set Format to FBX, Skin to With Skin (includes the mesh), Frames per Second to 30, and Keyframe Reduction to None. These settings give you the cleanest result to bring back into Blender.
Step 3 — Import into Blender
01
Import the FBX
In Blender, go to File → Import → FBX and select your downloaded file. Your character arrives with its Mixamo armature and the animation already baked onto it.
File → Import → FBX (.fbx)
02
Fix the scale
Mixamo exports at 100× Blender's default scale, so your character may appear enormous. Select it and scale down to 0.01, then apply the scale so it behaves correctly going forward.
S → type 0.01 → Enter, then Ctrl A → Scale
03
Preview the animation
Press Space to play. Your character should animate immediately. If it doesn't, check that the timeline range covers the animation — Mixamo clips are usually 30–250 frames depending on the motion.
Space Play / Pause
04
Blend multiple animations with the NLA Editor
Import additional Mixamo clips as separate FBX files. In the NLA Editor, stack their actions as NLA tracks to sequence or blend animations — idle into walk into run, for example.
Editor Type → Nonlinear Animation
