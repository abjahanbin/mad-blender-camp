Drawing Exercises

Drawing / 2D

Daily warm-up focused on visual effects in 2D. Draw a moment of impact, explosion, or motion — use speed lines, debris trails, impact flashes, and shockwave rings to communicate force without showing it directly. Every VFX element has a 2D equivalent: a spark trail is a line with a gradient, a shockwave is a circle with a blur. Train your eye before the computer does it.

Advanced Compositing

Animation

Day 6 introduced the compositor — today we go deeper. Render Passes split your scene into separate image layers (diffuse color, shadows, specular highlights, reflections, depth) that can be adjusted and recombined with full control after the render. Grade shadows and highlights independently. Composite characters against new backgrounds using alpha channels and the Alpha Over node. Use the Z-depth pass with a Defocus node to add cinematic depth of field entirely in post — no re-rendering needed.

Enabling Render Passes
01
Enable passes in View Layer Properties
In View Layer Properties (the camera-with-layers icon), open the Passes section. Enable Diffuse Color, Shadow, and Z-Depth at minimum. Each enabled pass becomes a separate output socket on the Render Layers node in the compositor.
View Layer Properties → Passes → enable as needed
02
Access passes in the compositor
Open the Compositor workspace and check Use Nodes. The Render Layers node now shows a socket for each enabled pass — drag a link from any pass socket to inspect it with a Viewer node, or pipe it into a separate processing chain.
Ctrl Shift Left Click on any node to preview
03
Grade shadows independently
Route the Shadow pass through an RGB Curves node. Darken it, tint it, or soften its edges, then combine it back with the image using a Mix node in Multiply mode. You can adjust shadow darkness and color without touching the rest of the scene.
Shift+A → Color → RGB Curves  ·  Shift+A → Color → Mix
04
Add depth of field with the Z pass
Route the Z (depth) pass into a Defocus node's Z socket, and the combined image into its Image socket. Set the Bokeh type and F-stop to control how strongly background elements blur. Tweak focus distance without re-rendering a single frame.
Shift+A → Filter → Defocus
05
Composite elements with Alpha Over
To place a 3D render over a background image, connect your render to the upper Image socket of an Alpha Over node and the background to the lower. The alpha channel of your render acts as the mask. Stack multiple Alpha Over nodes to layer multiple elements.
Shift+A → Color → Alpha Over

Visual Effects & Particles

Animation

Particle systems let Blender simulate and animate thousands of objects at once — embers drifting from a fire, sparks ricocheting off an impact, leaves scattered by wind, confetti falling through a scene. Set up an emitter object, configure the physics (gravity, air resistance, initial velocity), and control the look of each particle. Combine the rendered result in the compositor to add glow and motion blur that sell the effect as part of the scene.

Setting Up a Particle System
01
Add a particle system
Select the object that will emit particles. In the Particle Properties panel (the dotted-triangle icon), click the + button to add a new particle system. The default type is Emitter — the object will emit particles from its surface.
Particle Properties → + → Emitter
02
Set count, start, and lifetime
Under Emission, set the total Number of particles, the Start frame (when they first appear), the End frame (when emission stops), and Lifetime (how many frames each particle lives before disappearing). Start with low counts — 200 to 1000 — until the sim looks right.
03
Tune velocity and gravity
Under Velocity, set the Normal value to shoot particles outward from the surface. Under Field Weights, reduce Gravity to get a floatier, slower-falling effect — or crank it for a hard drop. Add randomness to Normal and Tangent for natural variation.
04
Render as objects or halos
Under Render, choose Object to use any mesh as the particle shape (great for leaves, debris, or coins), or Halo for simple lit spheres (sparks, dust, stars). For Object rendering, pick an instance collection containing your particle object.
Particle Properties → Render → Render As
05
Add glow in the compositor
After rendering, use a Glare node in the compositor to add bloom around bright particles — essential for embers, sparks, or neon. Use a Blur node to add subtle motion blur to fast-moving particles when the rendered frames look too sharp.
Compositor → Shift+A → Filter → Glare / Blur

Mixing & Editing

Sound Design

Bring all your recorded and designed audio together. Balance levels so nothing is too loud or buried, shape frequencies with EQ, add reverb and spatial width to give sounds depth and placement. Build a final audio timeline that supports your animation — tight enough to feel intentional, spacious enough to breathe.
